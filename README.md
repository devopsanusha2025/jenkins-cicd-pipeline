# Jenkins CI/CD Pipeline

This project demonstrates a declarative Jenkins CI/CD pipeline for automating build, test, Docker image creation, and deployment steps.

## Tech Stack
- Jenkins
- Git & GitHub
- Docker (build stage)
- Linux

## Pipeline Stages
1. Checkout source code from GitHub
2. Build application
3. Run unit tests
4. Build Docker image
5. Deploy application

## Usage
- Create a Jenkins Pipeline job
- Connect this GitHub repository
- Jenkins will automatically execute stages defined in the Jenkinsfile

## Outcome
- Automated CI/CD pipeline
- Reduced manual deployment effort
- Faster and reliable releases
